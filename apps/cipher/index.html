Cipher

Encrypt & Decrypt Messages

Use Cipher to send encrypted messages to friends & family. They can then decrypt it in Cipher too! Cipher is available to download on iOS, iPadOS and macOS.

Features

Encrypt

Cipher makes it super quick & easy to generate an encrypted message. Just enter your message in plain text and hit Encrypt. You can then send the scrambled message and encryption key to friends & family using the share button.

Decrypt

If you receive an encrypted message from another Cipher user, you can use the Decrypt function to view it. Enter the sent encryption key and the encrypted message, then select Decrypt. The message will then be decrypted and you’ll be able to view the original message!

Technologies

Cipher has been built using SwiftUI and CryptoKit.
