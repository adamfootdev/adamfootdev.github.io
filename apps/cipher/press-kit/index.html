Features

Encrypt

Cipher makes it super quick & easy to generate an encrypted message. Just enter your message in plain text and hit Encrypt. You can then send the scrambled message and encryption key to friends & family using the share button.

Decrypt

If you receive an encrypted message from another Cipher user, you can use the Decrypt function to view it. Enter the sent encryption key and the encrypted message, then select Decrypt. The message will then be decrypted and youll be able to view the original message!
