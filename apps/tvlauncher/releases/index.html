TV Launcher Releases

Version 3.0

3 December 2025

Liquid Glass

TV Launcher has been fully redesigned to support Liquid Glass and the latest OS version across all Apple platforms. This update also improves usability throughout the app and introduces a number of new features to improve the experience.

Home

Watch Now is now Home and features a redesigned layout that makes it quicker to access different categories. On iPad, Mac, Apple TV and Apple Vision Pro, a new sidebar shows a list of categories and apps for access to channels contained within them.

Watchlist

Up Next is now Watchlist which can be found in its own tab. The next programme’s delete button will now be selected when deleting the programme above too.

Top Shelf

Programmes currently on or starting within the next hour are now visible in the Top Shelf. This can be disabled in Settings > Top Shelf. The start & end time of a programme is now also visible in the Top Shelf.

More

As well as the main features, these are some of the other changes that are new or improved in version 3: • Added the ability to add multiple TV Guides. These can be reordered to influence how channels are sorted. • Custom links can be added to channels so you can launch channels via deep link in apps not currently supported by TV Launcher. • Added support for widgets on visionOS so you can quickly launch channels and see what’s on without opening the app. • The order of channels can be customised using a new priority value in settings. Channels will initially be sorted by associated TV Guide and then a default value but channels can now be set in a fully custom order. • Added the ability to add/remove apps supported for each channel and to change the associated guide for each channel. Channels can also be added without a guide too. • The programme details screen has been redesigned with an improved layout of the main actions. • When launching a channel, if an app is not installed, the website will now be launched instead. • Improved the launching of some apps/channels throughout the app. • Added the Open Programme in Apple TV action throughout the app so it is quicker to access programmes in Apple’s TV app. • Added support for widgets in CarPlay on iOS. • Pressing the play/pause button on a programme tile will now launch the channel. The select button will still respect the Default Programme Action setting. • Settings have been simplified to make ordering of channels easier and prevent channels being mistakenly hidden. • Programmes no longer appear in separate windows on macOS and are now contained within the navigation stack for improved navigation throughout the app. • Improved navigation throughout the app. • Improved the scrolling performance of the TV Guide on tvOS. For programmes that are longer, the title will now be displayed multiple times so it is easier to see the listing. • Channels that have no guide data are now hidden by default but can be re-added in Settings > Channels. • Updated available channels. • Updated the app icon to support Liquid Glass. • Fixed a bug where the time bar on the TV Guide may not update in real time.

Version 2.12

16 September 2024

Fall 2024 Update

• On the latest version of iPadOS, TV Launcher now uses the new tab bar style navigation structure. • Updated the layout of the toolbar items and combined the filter & sort menus. • Added dark & tinted app icons on iOS & iPadOS. • Added support for the Double Tap gesture on Apple Watch to add programmes to Up Next. • Programmes now open in a separate window on Mac. • Added TNT Sports & Premier Sports channels to open in NOW for Irish users. • Added Sky Sports UHD 1 & 2 on Apple TV. Please note the TV Guide does not support these channels and they can only be launched from Apple TV. Sky Sports Main Event UHD & Sky Sports F1 UHD will be automatically launched using the standard channels if you have a NOW Ultra Boost subscription. • Sky Sports Arena has been renamed to Sky Sports+. • Updated the availability of channels on Apple Vision Pro. • Tips now sync via iCloud so you’ll only see them on one device. • Added support for the latest version of iOS, iPadOS, macOS, tvOS, visionOS & watchOS.

Version 2.8

18 September 2023

iOS 17 Update

• Redesigned watchOS app for watchOS 10. • Existing Siri Shortcuts & Widgets now use AppIntents for iOS 17. • Added support for AppShortcuts so Shortcuts can be used without configuring first. • Added a Shortcut to fetch the currently airing programme for a channel. • Added a Shortcut to fetch the Now & Next status for a channel. • Added a Shortcut to fetch the TV Guide on a specific date for a channel. • Added a Shortcut to fetch programmes in Up Next and another to fetch programmes you have reminders set for. • Added Shortcuts to add or remove programmes from the Up Next queue. • Added Shortcuts to set or remove reminders for programmes. • Added a Find Channel Shortcut that filters available channels in the app. • Added Shortcuts to add or remove channels from Favourites. • Added Shortcuts show or hide channels. • Added a Shortcut for watching a programme - works similar to the Watch Channel Shortcut but accepts a programme instead. • Added an Inline Channels Widget. • Support for StandBy on iOS 17. • Support for Lock Screen Widgets on iPadOS 17. • Added tips using TipKit throughout the app. • Improved the loading of channels on the watchOS app. • Support for the Smart Stack on watchOS 10.
