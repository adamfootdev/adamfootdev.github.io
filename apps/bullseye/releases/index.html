Bullseye Releases

Version 2.1

15 January 2026

iPhone & iPad App

Bullseye is now available to download on iPhone & iPad! All features from the existing Mac app are available to use on a mobile device so you can manage Apple Ads campaigns on the go.

More

• Added the ability to adjust the Max CPT Bid for multiple keywords and ad groups at once from the inspector. • When adding an account, Terminal is no longer required to create keys - Bullseye does this for you. • Added support for pre-order metrics throughout the app due to upgrading to version 5.4 of the Apple Ads Campaign Management API. • Text fields now become active on some screens when they appear instead of needing to select the text field first. • Added the ability to copy values on the Campaign Group Details screen. • Added the option to hide the status bar from Settings. • Added the option to hide the inspector on app launch to Settings.
