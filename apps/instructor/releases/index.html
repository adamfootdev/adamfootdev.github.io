Instructor Releases

Version 7.0

23 February 2026

Redesign

Instructor has been redesigned with a new layout and intuitive new features to make managing your ADI business even easier!

Diary

Added a new Diary tab which shows a list of all upcoming lessons for your students.

Drawing Pad

Added a new Drawing Pad tab which adds the ability to draw on different map types or a whiteboard using an Apple Pencil or your finger. For roads you always come back to, snapshots can be saved for later and viewed again.

Dashboard

The student dashboard screen has been redesigned with a single screen combining lessons, progress and notes. Statistics such as hours learnt for and how many lessons have been booked will also show here.

Lessons

Multiple lessons can now be added to each student with the ability to mark a duration, set type (lesson, test or mock) and mark whether it has been paid for.

Test Dates

Added the ability to set test and theory dates for students with a countdown showing on their dashboard.

Progress Tracking

Subtopics can now be added to each topic for more detailed progress tracking. Progress tracking now uses percentages in blocks of 20% to track progress too. It is also now possible to reorder topics too.

Contact Details

It is now easier to contact a student by phone, text or email from the dashboard and there is a new option to open their address in Apple Maps for navigating.
